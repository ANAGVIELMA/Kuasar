Productos Principales

Kubernetes Clústeres

KVM es un servicio de instancias de cómputo gestionadas directamente por nuestros clientes. Este servicio se considera dentro del segmento IaaS.

Kuasar virtual machines es nuestro servicio IaaS ofrecido a nuestros clientes para desplegar máquinas virtuales aisladas en una VPC (Virtual Private Cloud)

Kuasar Virtual Machines

Creación de clúster de Kubernetes y gestión directa de sus recursos. Este servicio es un PaaS.
